# Tic-Tac-Toe in Python

Ein klassisches Tic-Tac-Toe-Spiel, das ich mit Python programmiert habe.

## Über das Projekt

Dieses Projekt ist eines meiner ersten größeren Python-Projekte. 
Dabei habe ich verschiedene Grundlagen kombiniert und ein vollständiges kleines Spiel entwickelt.

## Funktionen

- Spiel für zwei Spieler (X und O)
- Darstellung des Spielfelds in der Konsole
- Eingabe über die Tastatur
- Überprüfung auf Gewinner
- Erkennung von Unentschieden
- Verhindert das Überschreiben bereits belegter Felder

## Was ich dabei gelernt habe

Bei diesem Projekt habe ich unter anderem gelernt:

- Arbeiten mit Listen
- Verwendung von Funktionen (`def`)
- `while`-Schleifen und Spielabläufe
- Bedingungen mit `if`, `elif` und `else`
- Rückgabewerte mit `return`
- Verwaltung von Spielzuständen
- Strukturierung eines größeren Programms
